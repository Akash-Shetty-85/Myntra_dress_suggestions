Directory and File Description
backend/: Contains all backend-related files.

app.py: Main Flask application file.
model/: Directory for model-related scripts.
__init__.py: Makes the model directory a package.
predict.py: Contains the prediction logic.
train.py: Contains the model training logic.
static/: Directory for static files like CSS, JS if needed for backend rendering.
templates/: Directory for HTML templates if using Flask to render templates directly.
frontend/: Contains all frontend-related files.

index.html: Main HTML file for the web application.
css/: Directory for CSS files.
styles.css: Main CSS file for styling.
js/: Directory for JavaScript files.
app.js: Main JavaScript file for handling frontend logic.
data/: Directory for data-related files.

raw/: Directory for raw data.
images/: Directory for raw images.
processed/: Directory for processed data.
notebooks/: Contains Jupyter notebooks for data preprocessing and experimentation.

data_preprocessing.ipynb: Notebook for data preprocessing.
requirements.txt: List of Python dependencies.
